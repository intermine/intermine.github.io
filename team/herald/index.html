A passionate programmer who ended up at InterMine by chance following his interest in Clojure(Script). Is mostly versed in frontend development, but doesn't mind as long as he gets to write code in a fun language.

InterMine is open source (LGPL 2.1) and free to use. It's a software system that you can install on your own servers to make data available on the web.
